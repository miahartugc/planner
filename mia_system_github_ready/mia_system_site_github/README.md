# Mia System

Static HTML app for GitHub Pages.

## How to upload to GitHub

Upload the full contents of this folder, not just `index.html`:

- `index.html`
- `assets/avatar_square.png`
- `assets/shop_cat.png`

Then enable GitHub Pages from the repository settings.

The images are external files instead of embedded base64, so the page should not crash from a huge HTML file.
